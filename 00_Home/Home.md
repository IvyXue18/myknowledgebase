### 状态书影音地图

这不是普通片单/书单。
它记录的是：**我在什么状态下，什么作品对我有用。**

已完成的打勾；如果后面确认不适合我，可以直接移除。

### 怎么用

1. 先判断我现在是哪种状态
2. 再从对应清单里挑一部/一本
3. 看完后，如果值得留下，就补进 `04_Works/` 的正式作品卡

### 这组作品共同在回应什么

这批作品不是为了提供“人生答案”，而是帮助我在以下状态里重新安住：

- 一个人干着自己的事，但暂时看不到很大成果
- 生活半径变小，精神有点封闭、孤僻
- 想回家，但也知道“回家”不一定能真正解决安放问题
- 想做事，但沉不下来
- 需要一些低刺激、非鸡血的精神落点

它们共同的部分是：

1. **把注意力从大成果拉回小秩序**：吃饭、散步、做饭、打扫、写信、认真工作。
2. **不急着解决孤独，而是让孤独有形状**：承认一个人会空、会漂、会不确定。
3. **让人与世界重新发生微弱连接**：一顿饭、一封信、一个熟悉的店、一次短暂照看。
4. **让人重新相信普通生活也值得**：不是只有成功、热闹和被看见时，人才算活着。

### 状态清单 v2

#### 人物状态：封闭、孤独、看不到很大成果，但还想把手头的事做下来

这种时候不一定需要热闹，也不适合太强目标感、强逆袭、强社交的内容。
更适合看：一个人如何靠日常秩序、微弱连接和一件认真做的小事，把自己托住。

- [x] [[完美的日子]]｜电影
  - 关键词：秩序、日常、树影、一个人也能活出质地
  - 推荐理由：它不试图消灭孤独，而是展示孤独如何被稳定的日常动作托住。适合我在“一个人有点空，但又不想被鸡血推动”的时候看。
- [ ] [[小森林]]｜电影
  - 关键词：食物、四季、身体、乡村、慢慢活过来
  - 推荐理由：适合把我从脑子里的乱和空拉回身体。它提醒我：生活不是只有项目、收入、结果，还有吃饭、季节、手感和身体。
- [ ] [[海鸥食堂]]｜电影
  - 关键词：小店、异乡、低压力连接、清爽
  - 推荐理由：它像一个稳定的小空间，慢慢吸引合适的人靠近。适合我想从封闭里松动一点，但又不想被复杂关系打扰的时候。
- [ ] [[帕特森]]｜电影
  - 关键词：日常重复、诗、普通工作、观察力
  - 推荐理由：它很适合提醒我：普通日子里也可以有诗性。不是每天都要有大突破，一个人也可以在重复生活里保留精神纹理。
- [ ] [[山茶文具店]]｜书
  - 关键词：写信、代人表达、慢、温柔秩序
  - 推荐理由：它回应的是“很多话人自己说不清，需要有人帮他把心意整理出来”。这也和我正在探索的“帮人整理隐性上下文”有关。
- [ ] [[一个人的好天气]]｜书
  - 关键词：独居、漂浮、别扭、慢慢和生活磨合
  - 推荐理由：适合那种“不想振作，但也不想继续散掉”的状态。它不美化孤独，也不把孤独写成绝望，只是陪一个人慢慢适应生活。
- [ ] [[给青年诗人的信]]｜书
  - 关键词：孤独、创作、向内生长、精神骨架
  - 推荐理由：适合我想理解“孤独不是失败，而可能是长出自己的一部分”的时候读。比鸡汤更深，也更安静。

## 来源处理

未来原文剪藏进入 `09_Raw/_inbox/`。

### 状态定义

- `inbox`：刚剪藏，未处理
- `digested`：已经被吸收到知识页
- `ignored`：无吸收价值，只保留归档
- `needs_review`：AI 判断不确定，需要人复核

### 价值定义

- `high`：值得进入专题、人物、概念或作品页
- `medium`：保留索引摘要，可能进入某个页面
- `low`：保留原文，不主动专题化
- `unrated`：未判断

## Markdown 速查

Obsidian 支持 CommonMark、GitHub Flavored Markdown、LaTeX，并额外支持 Wikilink、嵌入、Callout、高亮等语法。快捷键可以在 **设置 → 快捷键 Hotkeys** 里搜索命令修改。

### 最常用格式

| 用途 | Markdown 写法 | 快捷键 |
| --- | --- | --- |

| 加粗 + 斜体 | `***文字***` | 可分别按加粗、斜体 |
| 高亮 | `==文字==` | 默认通常没有固定快捷键，可自定义 |

| 代码块 | 三个反引号包裹 | 默认通常没有固定快捷键 |
